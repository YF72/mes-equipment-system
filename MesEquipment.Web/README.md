# MES Equipment Web

Angular 21 frontend for the MES Equipment Management System.

For full project setup, backend instructions, and demo notes, see the root
`README.md`.

## Commands

```bash
npm install
npm start
npm run build
npm test
```

## Local URL

```text
http://localhost:4200
```

## Frontend Highlights

- Angular 21
- NgRx Store / Effects / Selectors
- RxJS
- Signals for local UI state
- Reactive Forms for machine list filters
- Functional auth interceptor
- Route guards
- Tailwind CSS
